то програма на Python, която преобразува числа (разделени със запетая и интервал) обратно в букви според зададеното съответствие:
python

def numbers_to_text(numbers_str):
    # Създаваме речник за съответствието между цифри и букви
    num_to_char = {
        '31': 'А',
        '2': 'Б',
        '3': 'В',
        '4': 'Г',
        '5': 'Д',
        '6': 'Е',
        '7': 'Ж',
        '8': 'З',
        '9': 'И',
        '10': 'Й',
        '11': 'К',
        '12': 'Л',
        '13': 'М',
        '14': 'Н',
        '15': 'О',
        '16': 'П',
        '17': 'Р',
        '18': 'С',
        '19': 'Т',
        '20': 'У',
        '21': 'Ф',
        '22': 'Х',
        '23': 'Ц',
        '24': 'Ч',
        '25': 'Ш',
        '26': 'Щ',
        '27': 'Ъ',
        '28': 'Ь',
        '29': 'Ю',
        '30': 'Я'
    }
    
    # Разделяме входния низ на отделни числа
    numbers = numbers_str.split(', ')
    text = []
    
    for num in numbers:
        if num in num_to_char:
            text.append(num_to_char[num])
        else:
            # Ако числото не е в речника, го добавяме както е (за препинателни знаци и др.)
            text.append(num)
    
    # Съединяваме буквите без интервали
    result = ''.join(text)
    return result

# Примерна употреба
input_numbers = input("Въведете числа разделени със ', ': ")
output_text = numbers_to_text(input_numbers)
print(output_text)

Как работи програмата:

    Създава се речник num_to_char, който съпоставя всяко число с неговата кирилска буква.

    Входният низ с числа се разделя на отделни числа чрез split(', ').

    Всяко число се проверява:

        Ако е в речника, се добавя съответната буква.

        Ако не е в речника (например препинателен знак), се добавя без промяна.

    Буквите се съединяват в низ без интервали.

Пример:

Ако въведете "31, 2, 3", програмата ще върне "АБВ".
Забележки:

    Програмата очаква числата да са разделени точно със ", " (запетая + интервал).

    Ако въведете невалидно число (което няма съответствие в речника), то ще бъде добавено към резултата без промяна.
